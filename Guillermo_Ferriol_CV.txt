AI-ASSISTED DEVELOPMENT
================================================================
Hands-on orchestrating AI coding agents in production workflows — not
just prompting, but engineering the systems around them.

Tools:      Claude Code, Codex, GitHub Copilot, OpenCode, Ollama (local LLMs)
Practices:
- Spec-Driven Development — treat a structured spec as the single source
  of truth and drive the agent to generate, test, and validate against
  it; focus on intent and acceptance criteria ("what") over manual coding.
- Harness Engineering — design the environment, constraints, and feedback
  loops around an agent (Agent = Model + Harness) so it ships reliable
  code at scale with less review toil.
- Loop Engineering — design autonomous act → observe → decide → repeat
  loops (ReAct pattern) with clear goals and stopping conditions so
  agents iterate and self-correct toward a target.
